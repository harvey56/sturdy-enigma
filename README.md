## spin-up instructions

### Go Backend API

This repository contains the backend service for the storytelling engine. It is built with Go and designed to be highly scalable, container-ready, and easy to deploy.

🚀 Quick Start

1. Prerequisites

Ensure you have the following installed:

    Go (version 1.25 was used here)

    Docker

2. Environment Setup

.env variables are defined in Cloud Run

The list of env variables is at the start of the main.go

3. Installation

`go mod tidy`

4. Running the Application

To start the server : `go run main.go` or F5
